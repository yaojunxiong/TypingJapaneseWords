Japanese Typing Tutor 说明书

1. 项目概述

JapaneseTypingTutor 是一款专为日语学习者设计的打字练习工具，旨在提高用户的打字速度和准确率，同时巩固日语词汇。

2. 项目目标

  • 通过打字练习强化日语词汇记忆
  • 提供多种练习模式，提高学习兴趣
  • 统计用户的输入速度和准确率
  • 支持自定义单词列表

3. 核心功能

3.1 基础打字练习

系统会显示日语单词，用户正确输入后进入下一个单词，同时统计 WPM（每分钟输入词数）和准确率。

3.2 JLPT 词汇练习

内置 N5 至 N1 词汇列表，自动筛选用户易错单词进行重点练习。

3.3 计时模式 & 挑战模式

设定时间限制（如 1 分钟）进行打字挑战，并统计用户输入的正确单词数量。

3.4 输入错误分析

记录用户的输入错误，统计易错单词，提供针对性的练习建议。

3.5 自定义单词列表

支持用户导入 Excel 文件，自定义单词列表，灵活调整学习内容。

4. 技术架构

  • 编程语言: Python
  • 前端技术: HTML + JavaScript
  • 数据管理: Excel 文件
  • 支持环境: Windows / macOS / Linux

5. 使用方法

5.1 运行方式

  • 下载项目文件
  • 打开 index.html
  • 开始进行日语打字练习

5.2 数据导入

  • 使用 Excel 进行单词管理
  • 通过 AppSheet 进行数据同步

6. 未来发展

  • 加入排行榜 & 挑战模式
  • 支持多用户数据存储，记录个人学习进度
